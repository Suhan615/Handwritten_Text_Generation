# Handwritten Text Generation using Character-Level RNN

This project implements a character-level Recurrent Neural Network (RNN) using LSTM to generate handwritten-like text. The model is trained on a text dataset and learns character patterns to generate new text sequences.

## Algorithm Used
- Character-level RNN (LSTM)

## Dataset
- Custom handwritten text dataset (`handwritten_text.txt`)

## Steps
1. Load and preprocess text data
2. Convert characters to numerical format
3. Train an LSTM-based RNN
4. Generate new text using a seed string

## Example Output
Dear rm ditvy,IDpsoiD,ypbrjmel.n aoshe.ja wthtmIsemblILunId ...

## Libraries Used
- Python
- TensorFlow
- NumPy
